# Brokerage Portal

Brokerage operations dashboard for the Reala portfolio monorepo.

## Highlights

- Workspace-aware command center for brokerage teams, agents, vendors, and client intake.
- Listing, order, job, approval, service catalog, billing, and agent management flows.
- Clerk/Supabase-ready architecture with route protection and organization-scoped access patterns.
- Engel & Volkers branded demo assets included for a realistic brokerage presentation.

## Run

```bash
pnpm --filter @reala/brokerage-portal dev
pnpm --filter @reala/brokerage-portal build
```
