# Reala Chatbot

AI intake and lead workflow app for Reala.

## Highlights

- Conversational interface for property, lead, and task workflows.
- Agent routing layer for managed handoffs and command-center actions.
- Database-backed chat state, tests, and production-oriented Next.js structure.
- Portfolio-ready example of turning product design flows into a functional product engineering surface.

## Run

```bash
pnpm --filter @reala/chatbot dev
pnpm --filter @reala/chatbot build
```
